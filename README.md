## SQL2O

Generate a Java back-end scaffolding quickly based on MySQL schema

### Features

 - project management and comprehension tool: Maven

 - frameworks: Spring Boot , MyBatis

### Csaffolding structure overview 

```
.
└── src
    └── main
        ├── java
        │   └── io
        │       └── laudoak
        │           ├── dao
        │           ├── model
        │           └── service
        │               └── impl
        └── resources
            └── mapper
```
